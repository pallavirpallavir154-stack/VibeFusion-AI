# VibeFusion AI

VibeFusion AI is an AI-powered entertainment platform that provides personalized music, video, and meme recommendations based on user preferences and mood selections.

## Features

* Music Recommendations
* Video Recommendations
* Meme Recommendations
* Gesture Recognition
* AI Analytics Dashboard
* User Profile Portal
* Rhythm Challenge Game

## Technologies Used

* HTML
* CSS
* JavaScript
* React
* TensorFlow
* MediaPipe

## Future Enhancements

* Voice Commands
* Advanced AI Recommendations
* Mobile Application
